# NowScore 后台规格对照清单

基于 `E:\Advanced File\NowScore後台功能規格文件.docx` 与当前代码实现整理。

## 已有主能力

- 内容管理：作者、计划单、首页模块、Banner、公告、AI 预测、AI 客服知识库、人工转接
- 用户与订单：用户列表、会员订单、内容订单、订单导出、退款、补单、状态修改
- 赛事同步：同步任务、最近运行记录、轮换计划、前台赛事数据接入
- 首页运营：Banner、焦点赛事槽位、模块运营位

## 本轮已落地到真实数据层

- 已新增真实数据库模型：
  - `CoinAccount`
  - `CoinLedger`
  - `CoinPackage`
  - `CoinRechargeOrder`
- 后台 `finance` 页已接入真实数据：
  - 真实球币套餐读取
  - 真实充值订单读取
  - 手动建单
  - 补单入账、失败、关闭、退款
  - 手动加币 / 扣币
  - 球币账户余额榜
  - 球币流水明细视图
  - 充值对账总览
  - 过期充值单清理入口
- 充值订单补单后会真实给用户账户入账，并生成球币流水
- 充值订单退款会尝试回退球币余额，并生成退款流水
- 后台人工调账会直接更新 `CoinAccount`，并写入 `CoinLedger`
- 已新增内部定时接口 `/api/internal/finance/recharge-expiry`，可由服务器 cron 自动关闭超时未处理充值单
- 前台已接入球币消费链路：
  - `/plans` 列表与详情已切换为球币优先购买
  - `/plans/[slug]` 详情支持球币解锁与结果提示
  - `/member` 会员中心支持球币余额、最近流水、最近充值订单展示
  - 球币解锁内容后会复用正式 `ContentOrder` 权限链路，已购状态即时生效

## 仍需继续深化的点

- 球币系统虽已落地真实 `球币账户 + 流水 + 充值订单 + 前台内容消费`，但还未补 `批量运营加币/扣币、可导出的财务对账报表、异常单批处理`
- 代理招商模块已补到真实数据层与后台可操作界面，但还未接 `邀请码归因到注册/充值、佣金自动结算流水、代理业绩报表、提现打款回单闭环`
- 系统安全模块当前提供管理视图，尚未落到可配置 RBAC、真实审计日志落库、告警渠道联动
- 报表中心当前为后台面板与导出语义补齐，尚未接完整 BI 聚合任务和异步导出任务中心
- 会员详情页仍需继续做单用户深挖页，补齐认证资料、球币流水、已购文章、代理关系、登录日志的真实详情
- 赛事补录和联赛管理目前为运营控制台视图，后续应补正式 CRUD 和持久化接口

## 后续优先级建议

1. 先补批量调账、可导出的财务对账报表和异常单批处理
2. 再做代理申请、邀请码归因、佣金流水、提现审核的数据库与接口
3. 再落审计日志、RBAC、监控告警的正式持久化
4. 最后把报表中心升级成真实聚合和异步导出
